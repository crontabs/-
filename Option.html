Crontab ข้าวโพดแห้ง

คำสั่งและ option ของ crontab

crontab filename

การนำเอาคำสั่ง crontab เข้ามาจาก ไฟล์อื่น

crontab -e

แก้ไข crontab ปัจจุบัน (ส่วนมากเราจะเพิ่มคำสั่งลงไปในนี้เลย)

crontab -l

ดูคำสั่ง crontab ทั้งหมดที่มีอยู่

crontab -r

ลบคำสั่ง crontab ที่มีทั้งหมด

crontab -u user

เป็นคำสั่งของผู้ดูแลระบบเท่านั้น เพื่อใช้ดู แก้ไข ลบ crontab ของ user แต่ละคน
